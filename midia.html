Apresentação do Projeto

ROTA ESCOLAR

Este projeto é uma aplicação inovadora desenvolvida para resolver o problema na vida dos estudantes em relação ao roteiro dos transportes escolares. Ele oferece rotas,horários,informa as paradas dos Ônibus e ETC.

informações do projeto

  • FACILITA A LOCOMOÇÃO DOS ESTUDANTES QUE DIARIAMENTE NECESSITAM DE HORÁRIOS E ROTAS PONTUAIS
  • O PROJETO FOI PENSADO COM O INTUITO DE AJUDAR A COMUNIDADE, PELO FATO DE SER ALGO QUE TAMBÉM NOS AFETA(criadoras do projeto) pois somos estudantes.
